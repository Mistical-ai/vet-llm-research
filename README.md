# Veterinary LLM Research Pipeline

This project aims to evaluate Large Language Models (LLMs) for their performance and accuracy on veterinary literature. The pipeline processes and analyzes veterinary papers from journals such as JVIM and JAVMA, extracting insights and evaluating model capabilities in the veterinary domain.

## Project Structure
- `data/`: Contains raw, processed, and mock data, as well as results and human review files. (Ignored by Git)
- `src/`: Source code for the pipeline, including utilities and the main entry point.
- `tests/`: Test fixtures and mock data for testing.
